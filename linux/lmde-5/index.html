來玩 LMDE 5

發表於 2022-04-05 文章字數： 1.3k 所需閱讀時間 ≈ 2 分鐘

前言

Linux Mint 目前在 DistroWatch 上的「網頁參觀排名」是第 3 名，也是我印象非常深刻的一支 Distro。

科普一下就可以知道，Linux Mint 除了基於 ubuntu 的版本之外，也有基於 Debian 的版本，而這篇貼文主要是我安裝基於 Debian 的 LMDE(Linux Mint Debian Edition) 5 後，使用一下下的快速回顧與感想。

安裝環境

  1. 處理器是 Intel(R) Core(TM) i5-7400(總之，是一部老機器)。
  2. 下圖是本次安裝 LMDE 5 的 VirtualBox 設定。

快速回顧與感想

  1. 介面是強項：LMDE 5 的介面設計而言，我個人非常喜歡，跟我目前使用的主力 Distro：MX Linux 相比，視窗調整上的鼠標感應範圍比較…合理，也得說 MX Linux 要調視窗大小，有點小考驗手眼協調與視力。下圖是 LMDE 在 MX Linux 上的 VirtualBox 運行的截圖。
  2. 相較基於 ubuntu 的 Linux Mint，LMDE 的上手難度似乎有高一點點：有關這一點，我是從預設的「輸入法框架」只有 XIM，而且還得自行去設定來思考的。原因無他，現在大部分的 Distro，再怎麼樣，都會依照各個語言在預設安裝時，把「輸入法框架」自動安裝並 on 上去，LMDE 則是要自行處理。如果今天一個非英文語系、對 Linux 有興趣，並且沒有接觸過 Linux 的人，想著要拿來當桌機使用，很有可能在安裝完不久，就決定退坑。…不過，我也得說，MX Linux 的輸入法框架更難搞，我花了一點時間上手後就愛不釋手。有時候，我會認為愛用 Linux 的人，不單單是熱情，而是多少有那麼點自虐的傾向吧！？

嘸蝦米輸入法的安裝與設定

  1. 以嘸蝦米而言，我比較推薦合法使用的 iBus 版本，但，如果要快速地使用，fcitx 是一個不錯的選擇。(下圖)

  2. sudo apt install fcitx-table-boshiamy(下圖)

  3. 【系統設定】→【輸入法】→【輸入法框架：】，設定為【Fcitx】，然後，重新啟動系統(重開機)。(下圖)

  4. 順利的話，就可以看到嘸蝦米輸入法。(下圖)

後記

  1. 本次安裝使用的 ISO 檔版本是 lmde-5-cinnamon-64bit.iso。
  2. Download LMDE 5 - Linux Mint
  3. 行易有限公司 :: 嘸蝦米輸入法
  4. LMDE 5 在介面上另外一個值得提的優點，就是類 Windows 的「開始」選單。我想，不得不說，除了少數幾個國家外，Windows 應該都是不少人第一個接觸的作業系統，習慣 Windows 的介面使用邏輯，再切換到不一樣的介面，如：macOS，多半會有一些不習慣。所以，我認為類 Windows 的介面設計算是一個優點，只是，骨子裡的靈魂不同，系統維護方式不一樣，要自己安裝程式、驅動程式等等，還是不太一樣，使用者能夠持續多久仍是一個難題。而我的觀察是即使已經走過那麼多年的發展，這個難題目前的解決之道，主要還是在於使用者有沒有需求，或是說能否在保有熱情的狀況下，持續使用 Linux。
