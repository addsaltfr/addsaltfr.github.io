在 Linux Lite 6.6 上安裝中文輸入法(含嘸蝦米輸入法) feat. Fcitx5

發表於 2023-09-22 文章字數： 778 所需閱讀時間 ≈ 1 分鐘

前言

本編文章，主要是紀錄 Fcitx5 搭配中文輸入法 (含嘸蝦米輸入法) 的服用過程。

安裝過程

  1. 先下指定安裝 Fcitx5 (含嘸蝦米輸入法的表格檔)，並且將系統的輸入法框架設定為 Fcitx5。

  2. 重新啟動後，整個桌面環境的下右下角，工作列會出現一個小鍵盤，點右鍵開啟選單後，再點「設定」，進入設定畫面，將嘸蝦米輸入法新增進來。

  3. 完成設定後，就可以同時享用新酷音輸入法與嘸蝦米輸入法。

後記

  1. 本次安裝使用的 ISO 檔版本是 linux-lite-6.6-64bit.iso。
  2. Linux Lite Easy to Use Free Linux Operating System
  3. Free Operating System - Linux Lite
  4. 在 Linux Lite 6.2 時，我曾經紀錄過以 fcitx(4) 服用中文輸入法，但是，並不是很順利，來到 Linux Lite 6.6 時，我特別下幾個指令觀察一下，發現自從我近幾代的 Linux Lite 在中文輸入法框架的選擇順序上，應該會是 iBus → Fcitx5 → fcitx(4)。
