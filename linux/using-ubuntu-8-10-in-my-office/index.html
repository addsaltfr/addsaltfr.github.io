嘗試將 ubuntu 8.10 帶入辦公環境

發表於 2008-11-25 文章字數： 772 所需閱讀時間 ≈ 1 分鐘

在我的辦公室裡面，全部都是 Windows，然而要讓勇敢的山羊在這個環境裡面勇往直前並不困難，我分析出以下幾個需要面對的問題：

  1. 沒有複雜的伺服器服務，如 Active Directory 等。
  2. 使用軟體單純，大部分都泡在 Office 軟體裡面，偶爾需要影像處理軟體也容易解決，字型的部分也容易解決。
  3. 設定印表機，以往要在 Linux 設定印表機，可說是難上加難，勇敢的山羊嗅覺靈敏，已經可以自動偵測。

想好，就一步一步來，我的選擇是先搞定硬體，也就是先安裝好印表機，再來是軟體，也就是 Office，Openoffice 我用不慣，最近又在趕報告，所以還是得喝點酒（Wine HQ），裝瘋賣傻在 Microsoft Office 上頭工作。

字型的部分，我的解決步驟如下：

  1. 先把 Windows 所在的硬碟掛載進來。
  2. 把所有的 TrueType 字型全部丟到 /usr/share/fonts/truetype/ 裡面。
  3. 再下指令 sudo fc-cache -f -v，搞定。

使用一個早上後，大概就是 Word 在列印時容易掛點外，沒什麼太礙事兒的部分，只是我並不滿意，畢竟還是會影響到我工作的心情與效率，況且 E-Mail、ie Only 的網頁都還沒有解決呢！

順便，也談點嚴肅的話題，就是有些伙伴們在進入 Linux 世界後都會問的：「怎麼 Linux 不能像 Windows 這樣、那樣的？」

基本上我的看法是 Linux 在桌面運用的部分近幾年來有明顯的進步，或多或少會去吸取其他系統的優點，這當然也包括 Windows，然而要把 Linux 當成 Windows 來用，無論是對 Linux 或是對 Windows 都不公平。曾經我也走過這段迷思，而今已曉得自己為什麼要用 Linux：絕不是討厭微軟，而是自由的精神。

你呢？
