使用 Windows Vista 上的 wubi 安裝 Ubuntu 8.10

發表於 2008-11-05 文章字數： 663 所需閱讀時間 ≈ 1 分鐘

如果不是因為日前乙部壞掉的光碟機，讓 wubi 於 Windows XP 上安裝 ubuntu 8.10 不成，得到的結果就不會像現在，把蒼鷺放飛…

由於我之前安裝 ubuntu 8.04 的關係，執行 wubi 後，就先要我把已經在機子上的 ubuntu 移除。

移除後，wubi 並不會進一步把新的 ubuntu 安裝上去，所以我就自己再去執行乙次。在 ubuntu 8.04 隨附的 wubi 若要在 Windows Vista 順利安裝，必須在硬碟上執行程式，否則最後會掛點，今次我直接執行光碟片裡面的 wubi，並未遭遇此一問題，可見已有明顯改善。

重新開機進入 ubuntu 8.10 後，有檢查安裝…的過程，我試著拍下來，可是再重新開機後就不見蹤影。而 X-BLUE 的無線網卡是 Broadcom 43XX 系列，無法像 Intel 系列那樣安裝完就立即可以使用（至少 Fedora 現在都是如此），所以後續有些套件需要從網路下載補完的，在沒有網路加持的前提下全然沒有安裝，尤以語系支援與硬體驅動程式最明顯。

ubuntu 8.04 時的 wubi 即使在安裝時選擇繁體中文語系，安裝後 apt 的 sources.list 裡面指向的伺服器全都在對岸，因此在安裝語系套件時，我特別看一下詳細的下載內容，全部都來自 TW，速度快令人爽。

把嘸蝦米表格檔擺好，開始愉快地使用 ubuntu 8.10 是一定要的。

先跟大伙分享至此，安裝完我還發現除了對無線網路的支援，還有其他好玩的東西，先玩再說囉！
