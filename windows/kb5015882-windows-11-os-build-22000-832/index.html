[版本更新] 快速回顧 微軟發布 KB5015882 (Windows 11 作業系統組建 22000.832)

發表於 2022-08-07 文章字數： 628 所需閱讀時間 ≈ 1 分鐘

前言

7 月 21 日，微軟針對 Windows 11 釋出 KB5015882 (作業系統組建 22000.832) 的更新。

更新前內容概覽

  1. 官方說明文件：2022 年 7 月 21 日 - KB5015882 (作業系統組建 22000.832) Preview，說明文件提到這支更新的重點。

  2. 這支補丁的更新重點是「非安全性更新包含品質改良功能」，而微軟也特別用斜體字強調幾項重點。

  3. 主要變更的部分，是「讓您選擇在開啟專注輔助時接收重要通知。 專注輔助就像是隱藏通知的[請勿打擾] 模式。」以及「如果您的裝置符合資格，可讓您選擇在第一次啟動 Windows 時更新至較新Windows 11版本。」

後記

  1. 數算起來，從 7 月 21 日到我真的把更新安裝上去，少說也讓我考慮個 1 個星期以上。
  2. 無論是哪一項新增功能並沒有吸引到我，其中，關於「如果您的裝置符合資格，可讓您選擇在第一次啟動 Windows 時更新至較新Windows 11版本。」這項更新，與其說是要給使用者，還不如說是微軟更想要使用者做出比較是微軟所期待的決定(希望我沒有理解錯誤，因為，我不是很確定我自己有沒有讀懂說明文件，這回連繞去讀英文版的氣力也懶得花)。
  3. 今次的補丁，與前次提到的 KB5015814 似乎在網路上都有人分享，都有同樣的災情發生：微軟認了 Windows 兩版本更新出包！狂按「開始選單」都沒有反應 - 自由電子報 3C科技，我個人更新後，使用至今是沒有碰到，也不希望碰上。
