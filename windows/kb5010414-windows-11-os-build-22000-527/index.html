[版本更新] 快速回顧 微軟發布 KB5010414 (Windows 11 作業系統組建 22000.527)

發表於 2022-02-18 文章字數： 654 所需閱讀時間 ≈ 1 分鐘

前言

今天來看 KB5010386 (作業系統組建 22000.493) 這一支補丁。

更新前內容概覽

我先讀過 2022 年 2 月 15 日 —KB5010414 (OS 版本 22000.527) 預覽 裡的說明，而這次的說明，我認為還算滿清楚的。

「新增功能！」是我特別注意到的部分，其中又以工作列的調整，最吸引我的注目。

如果工作列在中央對齊，則新增天氣內容至工作列左側。 當您將游標停留在天氣上時，畫面左側會顯示小工具面板，當您停止將游標停留在該區域上時，就會消失。

更新後的觀察

我認為工作列在中央對齊後，把一個後 Windows 10 時期放進工作列的小工具面板，移到最左邊是一個滿不錯的做法。坦白說，從我使用 Windows 11 之後，只要是工作列在中央對齊，我就會覺得工作列右邊好擁擠，而現在有這樣的調整，真的還滿不錯的。

儘管微軟做了調整，左右兩邊的「東西」，在量的部分，還是有點失衡，對我來說，如果左右兩邊的「東西」都可以自行調整的話，就更棒了。

Microsoft Teams(微軟團隊)是我很少用的 App，就…決定先把「自動啟動 Teams」關掉，至於它升級的部分，對常用的人，或許會覺得還不錯用。

<

後記

以往，我不會特別注意 Windows 更新的補丁，通常會注意到的時候，多半是釋出補丁之後，多數人補壞了才知道，要釋出穩定的補丁，還真的不是一件容易的事。

這次的更新，除了上述提到的 2 點之外，官方總共整理出 7 點，有興趣的伙伴，不妨在更新前讀一下，再決定要不要在這次更新。
