[版本更新] 快速回顧 微軟發布 KB4023057 (更新 Windows Update 服務元件)

發表於 2022-02-19 文章字數： 664 所需閱讀時間 ≈ 1 分鐘

前言

撰寫這一篇貼文時，我還沒有按下「立即安裝」，因為，我認為有必要花了一點點時間弄懂官方文件。

官方文件閱讀心得

(↓)今天要看的是 KB4023057 這支更新，繁體中文版的官方文件傳送門，請往 KB4023057：更新Windows更新服務元件 這邊走。我自己是比較少在讀微軟的官方文件的，一開始我真的讀不懂 KB4023057 究竟更新了什麼，下面我特別標示起來是在我讀完英文版的說明之後，才搞懂的。

(↓)英文版的官方文件傳送門，請走 KB4023057: Update for Windows Update Service components 這邊。

(↓)翻譯好，似乎不是那麼容易的事，或許，依循 Windows 系統上的名稱：「Windows Update」，可能就比較不容易弄錯。

(↓)KB4023057：更新Windows更新服務元件 的附註裡，這一點說明，句子感覺有漏字。…如果是我猜的「進一步瞭解」，那還真的掉了不少字。

(↓)循著附註提供的傳送門，走到 KB5001716：更新Windows更新服務元件 這一篇官方說明，…呃，「Windows更新Windows」的意思是？？？

後記

KB4023057 這支補丁，應該是要「更新 Windows Update 服務元件」，有點可惜的是…，無論是繁體中文或是英文，都看不太出來到底更新了什麼(猜想是「新的 Windows Update 使用者介面」)，所以，我最後的決定是先不要把這支補丁打上去。

如果有伙伴知道這支補丁的更新細節，歡迎在本篇貼文回應，謝謝。
