在 ubuntu 21.04 上輕鬆安裝嘸蝦米

來試試在 ubuntu 21.04 快速安裝 fcitx 並啟用嘸蝦米。

fcitx 在 ubuntu 21.04 並不是預設的中文輸入平台，不過，仍然可以在幾個步驟內就搞定。

  1. sudo apt-get install fcitx-table-boshiamy
  2. 一個指令，就讓相依的套件都安裝進來。
  3. 進入系統的「地區和語言」。
  4. 點進「管理安裝的語言」，把「鍵盤輸入法系統」改成 fcitx。
  5. 「登出再登入」，或是「重開機」。
  6. 順利的話，重新進入系統，就會看到工作列上面，多出了一個鍵盤，而且已經把嘸蝦米安裝進來。
  7. 使用 Ctrl+空白鍵(Space)，切換到嘸蝦米時，畫面上會跳出被框框包圍住的藍色嘸蝦米，同時，在右上角工作列那邊，原本的「鍵盤」圖示，也變成紅色的「嘸」。
